Feeds

In folderul feeds - care este un volum atast containerului de docker - vei gasii fisierle de feeds


Configuratia este pe un webserver de nginx - si se gaseste in container la calea usr/share/nginx/html


Pt a vedea un feed file acceseaza http://{CONTAINER_IP}/{NUME_FEED}


Comanda de docker pt a obine ip-ul containerului: docker inspect -f '{{range.NetworkSettings.Networks}}{{.IPAddress}}{{end}}' feeds


Containerul se numeste 'feeds'
